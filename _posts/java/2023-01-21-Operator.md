## 연산자
### 1. 정의
- 프로그램에서 데이터를 처리하여 결과를 산출하는 것을 의미한다.
- ex) x + y
  - x, y : 피연산자, + : 연산자


### 2. 종류
- 연산자별로 산출되는 값의 타입이 다르다.
- 피연산자의 수는 연산자를 사용할 때 필요한 피연산자의 개수를 의미한다.
- 증감, 비교, 논리, 대입, 삼항 연산자는 자주 사용된다.
